## 프로젝트 소개

- RealSense 카메라를 이용한 신체 비대칭 측정 프로그램
- 사람의 어깨 비대칭 정도, 안면 비대칭 정도, 안면 마비 정도를 알려주는 프로그램
- 매일 자신의 신체를 측정하고 라이프로그 그래프로 변화를 알 수 있음
